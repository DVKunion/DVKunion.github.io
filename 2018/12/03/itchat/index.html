ItChat

Itchat 使用笔记

0x00- - 前言

一直想做个自动回复性质的机械人，结合近期一直使用的Python就顺藤摸瓜的搜到了这个神奇的开源API。
这里是itchat-github，之后打算观察一下源码参考一下实现原理。
https://github.com/littlecodersh/itchat
在放一个官方的教程:
https://itchat.readthedocs.io/zh/latest/tutorial/tutorial0/

0x01- - 安装

itchat 可以直接使用pip安装:我使用的是Python3的版本。

pip3 install itchat

即可安装成功。

安装成功后，迫不及待的贴一段来运行一下:

import itchat
itchat.auto_login()
itchat.send('Hello, filehelper', toUserName='filehelper')

运行，手机扫码登录，文件传输助手收到了我们发送的信息。

0x02- - 初级使用

导包

第一步导包，常规操作。

import itchat

登录

之后是登录:

itchat.auto_login()

会生成登录二维码，若不想每次都扫码登录，可以带一个参数:

itchat.auto_login(hotReload=True)

测试发现，该登录方式会生成一个pkl的数据文件，从而直接跳过登陆的过程。

收取消息消息

itchat 在接受消息时需要根据要接受的消息类型进行注册;在获取对应的消息时会调用该函数:

当要接受一个文本消息时:

import itchat

@itchat.msg_register(itchat.content.TEXT) #这里就是注册操作.
#通过装饰符@将下面函数注册为文本消息的函数。
def print_content(msg):
    print(msg['Text'])

itchat.auto_login()
itchat.run()

可以通过修改注册类型信息即可收取不同的消息内容：

  1. 图片使用："itchat.content.PICTURE"
  2. 语音对应："itchat.content.RECORDING"
  3. 名片对应："itchat.content.CARD"
发送消息

发送消息可以发送文本、图片、文件等等，这里用文本为例:

itchat.send('Message Content', 'toUserName')

其中第一个参数为发送的文本消息，第二个参数㘝
将发送与接收同步一下，可以简单地做到：在收到消息时候回复简单的话语。手写一下进行尝试：

import itchat

@itchat.msg_register(itchat.content.TEXT)
def print_content(msg):
    return 'got it !'

itchat.auto_login()
itchat.run()

会发现：每一条消息都会自动的被回复"got it!"
