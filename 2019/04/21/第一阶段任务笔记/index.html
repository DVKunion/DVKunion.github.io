0x01- - 2019/4/21 第一周

第一周主要让大家能够了解和学习到一些 web 服务器的搭建，之后大家可以搭建自己的博客或是自己搭建本地靶机环境。

Lamp

首先放上我当时学习Lamp的网课连接：https://www.imooc.com/learn/170
讲的足够详细，如果你耐心看完了这个网课，这一部分的内容就可以直接略过了。

网课里面用的是Ubuntu 12.04，版本有一些年代感了，也可以选择更新一点的Ubuntu 16.04，但是可能有些操作会有一些微小的差别。
这里面涉及到一些Linux操作系统的操作…由于目前大家基础不同，对这一部分不了解的同学尽量自己去百度或者寻找网课独自补习一下。

那么，什么是Lamp呢？从名字上来说，他就是 Linux+Apache+Mysql+Php 一系列的软件；具体一点说，就是随着计算机 web 方面的不断发展，人们通过不断实践构造出的一套应用于动态网站或者服务器的解决方案。

可以这样理解，Lamp可以当成是一种方法，用于解决想要搭建动态网站这类问题的方法。

从 Lamp 的名字上来看，就可以知道要学习的内容有哪些：

  • Linux
  • Apache
  • Mysql
  • Php

我们需要做的就是在本地先运行一个虚拟机，安装一个Ubuntu 12.04的系统或是更高：这里注意，下载镜像的时候要区分是服务器版还是个人用户版，两者的区别是服务器版是没有桌面化环境的，只有命令行环境，所有操作只能靠命令符。

同时你也可以去腾讯云或者阿里云购买一个学生机，百度一下学生机，大概10元一个月，他会提供给你一个可以通过IP访问到的云主机，可以当做是一个远程的虚拟机。阿里云可能有集成化的Lamp安装选项，但是建议还是自己动手安装一遍走一遍流程，印象会更加深刻。

有了虚拟机之后我们就可以按照网课的步骤一步一步的进行安装和配置。

这里为了大家方便，我把视频中的一些指令放了出来方便大家复制，但是课程还是要看的哦。

$ sudo apt-get install apache2 
$ sudo apt-get install php5
$ sudo apt-get install mysql-server
$ sudo apt-get install php5-mysql
$ sudo apt-get install apache2 php5 mysql-server php5-mysql
$ sudo tasksel install lamp-server
#若安装php7.0需要手动安装libphp7.0.so
$ sudo apt-get install libapache2-mod-php7.0 -y

最后要达到的要求是，使用本地虚拟机的同学，可以通过物理机的浏览器去访问你虚拟机搭建的页面；云主机的同学可以使用销售商给你的公有IP在浏览器访问到你所搭建的页面。

搭建自己的博客

说实话，写博客其实要浪费很多的时间和精力，那为什么我还要推荐大家去做这个事情呢？

首先，写博客其实就是在把你所学习到的东西进行一次复述，所以在写博客的时候切记不要复制粘贴别人的内容,在复述的过程中你就很容易发现自己对哪个知识点存在一知半解的情况，然后进行巩固学习。

其次，写博客也是在做笔记，在以后用到的时候可以回头进行回顾和复习。

再有，博客也方便与大家交流和学习，互联网之所以发展的速度飞快就是在于他的共享精神以及极客精神。博客可以作为大家交流学习的一个载体。

最后，搭建出一个让自己看着心情“愉悦”的环境，会不会感觉学习都有动力了?比如我的看板娘Orz…(´▽`)

对于博客我不做太多的要求，主要是写作人自己觉得方便即可。

所以我希望大家能去弄一个小博客。

这里我给大家一些常见的博客方案，然后大家可以百度一下搭建方法，并不需要太麻烦的过程，我就不在这里复制粘贴搭建的过程了。

懒人方案：CSDN、Cblog、简书 等常见博客网站，申请账号即可。

免费方案：HEXO+github。这样作出来的界面是静态的，然后要求有一个github的账号。所需要的全部免费。

其他方案：WordPress+服务器 等等。需要有一台自己的云服务器。

还有很多其他的方案，大家如果搜索到其他的好的博客搭建方案也可以在群里分享出来鸭。
